# Tasheel API Documentation

Welcome to the Tasheel API documentation. This guide provides all the necessary details for merchants to integrate with the Tasheel platform.

## Available Documentation

### 1. [Basic Merchant API](./tasheel_api_documentation.md)
This document explains how to:
- Create orders (carts) for customers.
- Check the status of orders.
- Manage carts, including deletion and status updates.

### 2. [User Authorization API](./tasheel_api_documentation_v2.0.md)
This document focuses on:
- Authorizing users to complete payments directly from the merchant platform.
- Handling user authentication and authorization flows using OTP-based verification.
- Managing secure payment processes, including listing saved cards, setting defaults, and confirming payments.
